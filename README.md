This plugin overrides the auth plugin for Coral Talk comments.

To install,
- Install this in the /plugins dir
- remove `talk-plugin-auth` from `plugins.default.json`
- add `talk-plugin-dm-auth` to `plugins.json`

Sample plugins.json:

```
{
  "server": [
    "talk-plugin-dm-auth"
  ],
  "client": [
    "talk-plugin-dm-auth"
  ]
}
```
